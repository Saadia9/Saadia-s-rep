Coding Languages I'm learning:
  • HTML

  • JavaScript

Basic Structure:

  • HTML: Defines an HTML document.
  • Head: Contains meta-information about the HTML document.
  • Title: Sets the title of the HTML document (shown in the browser's title bar or tab).

Text Formatting:

  • p: Defines a paragraph
  • b or Strong: Defines bold text
  • i or em: Defines italic text
  • U: Defines underlined text (though not commonly recommended for use due to accessibility concerns)

Links and Navigation:

  • a: Defines a hyperlink
  • nav: Defines navigation links
  • ul: Defines a list of unordered items
  • ol: Defines a list of ordered items
  • li: Defines individual items within both ul or ol

Images and Multimedia:

  • img: Contains an image
  • video or audio: Contain a video or audio

Tables:

  • table: Defines a table
  • tr: Defines a table row
  • th: Defines a table header
  • td: Defines standard data cells within a table row

Style and Scripting:

  • style: Contains CSS styles within an HTML document.
  • script: Contains JavaScript or other scripting languages.

Selectors

  • (*, element, .class, #id, :pseudo-class, [attribute], etc.): Selectors are not tags themselves but are fundamental to CSS. They are used to target HTML elements based on various criteria such as element type (div, p, a), class (.), id (#), attributes ([type='text']), and pseudo-classes (:hover, :focus, :nth-child(n)).

    • Properties and Values:

  • CSS properties (property-name) and values (value) define how elements are styled. Some common properties include color, font-size, background-color, border, padding, margin, width, height, display, position, float, text-align, font-family, line-height, etc.

    • Box Model Properties:

  • Properties that define the dimensions and spacing of elements within the box model: width, height, padding, margin, border.

    • Text Styling Properties:

  • Properties that control text appearance: color, font-family, font-size, font-weight, line-height, text-align, text-decoration, text-transform, letter-spacing, word-spacing, etc.

    • Background and Border Properties:

  • Properties related to element backgrounds and borders: background-color, background-image, background-size, background-position, border, border-radius, box-shadow, outline, etc.

    • Variables (var, let, const):

  • Used to store data values.

    • Functions (function, Arrow Functions (=>)):

  • Used to define reusable blocks of code.

    • Arrays (Array):

  • Used to store multiple values in a single variable.

    • Objects (Object):

  • Used to store keyed collections of various data and more complex entities.

    • Conditional Statements (if, else, else if):

  • Used to perform different actions based on different conditions.

    • Loops (for, while, do...while):

  • Used to repeatedly run a block of code.

    • DOM Manipulation (document):

  • Used to interact with HTML elements and modify their content, attributes, and styles.

    • Event Handling (addEventListener):

  • Used to respond to user interactions with web pages.

    • Promises (Promise):

  • Used for asynchronous programming, representing a value that may be available now, in the future, or never.

    • Modules (import, export):

  • Used for organizing and reusing JavaScript code.
